# Smart Vision-Based Human Monitoring System

Overview

This project is a real-time computer vision system built using OpenCV. It detects human faces, monitors user presence, and demonstrates basic attention tracking using a webcam. The system is inspired by real-world ECE applications such as driver monitoring and surveillance systems.

Features

* Edge Detection using Canny Algorithm
* Face Detection using Haar Cascade Classifier
* Real-time Webcam Face Detection
* Human Presence Monitoring
* Basic Attention Tracking (face movement)

Technologies Used

* Python
* OpenCV

Project Structure

Image-Processing-Project
edge_detection.py
face_detection.py
webcam_face_detection.py
haarcascade_frontalface_default.xml
sample.jpg

 How to Run:

1. Install dependency:
   pip install opencv-python

2. Run edge detection:
   python edge_detection.py

3. Run face detection:
   python face_detection.py

4. Run webcam detection:
   python webcam_face_detection.py
